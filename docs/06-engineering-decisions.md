| Decision               | Reason                                                     |
| ---------------------- | ---------------------------------------------------------- |
| Express.js             | Lightweight, simple REST API for the assignment            |
| Vanilla JS             | Meets assignment constraint; no frontend framework         |
| OpenAI API             | Reliable LLM with good conversational quality              |
| No LangChain           | Explicit assignment requirement                            |
| Markdown Persona Files | Easy to edit, version control, and separate from code      |
| Stateless Backend      | Simpler architecture; frontend owns conversation history   |
| No Database            | Conversation is session-based; persistence wasn't required |
| Docker Compose         | Consistent local development and VPS deployment            |
| Nginx                  | Reverse proxy and SSL termination                          |
